# OnePS V1 Website

Official Version 1 website for OnePS, One Point Solutions.

The site uses the supplied official OnePS logo as the master brand asset. Utility favicon and manifest icon sizes are derived from that source without redesigning the mark.

## Structure

- `index.html` - Home
- `about.html` - Company story, mission, vision, principles
- `services.html` - Cloud engineering services
- `knowledge.html` - Searchable knowledge hub and learning paths
- `architecture.html` - Reference architecture center
- `projects.html` - Project showcase
- `contact.html` - Consultation contact page
- `404.html` - Not found page
- `css/` - Design system, components, utilities, animation, responsive layers
- `js/` - Progressive enhancement scripts
- `images/logo/` - Official logo and derived app icons

## Local Preview

Run a static server from this folder:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
